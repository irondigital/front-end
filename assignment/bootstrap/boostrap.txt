Q1. What is Bootstrap?

Bootstrap is a free, open-source front-end framework used to design responsive and mobile-first websites quickly using prebuilt CSS and JavaScript components.

Q2. How do you include Bootstrap in your project?

By adding the CDN links inside <head>:
<!-- Bootstrap 5 CSS -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">

<!-- Bootstrap 5 JS (with Popper) -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>

What is a container class in Bootstrap?

<div class="container">Fixed-width centered container</div>
<div class="container-fluid">Full-width container</div>


Q4. How do you create a responsive grid with 3 equal columns?

<div class="container">
  <div class="row">
    <div class="col">Column 1</div>
    <div class="col">Column 2</div>
    <div class="col">Column 3</div>
  </div>
</div>

Q5. How do you make a button using Bootstrap?

<button class="btn btn-primary">Primary Button</button>


Q6. Difference between d-none and invisible in Bootstrap?

d-none: Hides element and removes it from layout.

invisible: Hides element but keeps layout space.

Q7. Create a responsive navigation bar in Bootstrap.

<nav class="navbar navbar-expand-lg navbar-dark bg-dark">
  <div class="container-fluid">
    <a class="navbar-brand" href="#">Brand</a>
    <button class="navbar-toggler" data-bs-toggle="collapse" data-bs-target="#navMenu">
      <span class="navbar-toggler-icon"></span>
    </button>
    <div id="navMenu" class="collapse navbar-collapse">
      <ul class="navbar-nav ms-auto">
        <li class="nav-item"><a class="nav-link" href="#">Home</a></li>
        <li class="nav-item"><a class="nav-link" href="#">About</a></li>
      </ul>
    </div>
  </div>
</nav>

Q9. How do you create a modal in Bootstrap?

<button class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#myModal">Open Modal</button>

<div class="modal fade" id="myModal">
  <div class="modal-dialog">
    <div class="modal-content">
      <div class="modal-header"><h5 class="modal-title">Modal Title</h5></div>
      <div class="modal-body">This is a Bootstrap modal.</div>
      <div class="modal-footer"><button class="btn btn-secondary" data-bs-dismiss="modal">Close</button></div>
    </div>
  </div>
</div>


Q10. What are Bootstrap breakpoints?

Bootstrap breakpoints define screen sizes:

sm ≥ 576px

md ≥ 768px

lg ≥ 992px

xl ≥ 1200px

xxl ≥ 1400px

Q11. Create a responsive image in Bootstrap.

Q12. How to align text center in Bootstrap?

<p class="text-center">This text is centered.</p>


Q13. What class is used for spacing (margin & padding) in Bootstrap?

<div class="p-3 m-2 bg-light">Box with padding & margin</div>


Q14. How do you create a responsive table in Bootstrap?

<div class="table-responsive">
  <table class="table table-bordered">
    <thead><tr><th>#</th><th>Name</th></tr></thead>
    <tbody><tr><td>1</td><td>John</td></tr></tbody>
  </table>
</div>

Q15. How do you create a carousel (slider) in Bootstrap?

<div id="carouselExample" class="carousel slide" data-bs-ride="carousel">
  <div class="carousel-inner">
    <div class="carousel-item active"><img src="img1.jpg" class="d-block w-100"></div>
    <div class="carousel-item"><img src="img2.jpg" class="d-block w-100"></div>
  </div>
  <button class="carousel-control-prev" data-bs-target="#carouselExample" data-bs-slide="prev">
    <span class="carousel-control-prev-icon"></span>
  </button>
  <button class="carousel-control-next" data-bs-target="#carouselExample" data-bs-slide="next">
    <span class="carousel-control-next-icon"></span>
  </button>
</div>
