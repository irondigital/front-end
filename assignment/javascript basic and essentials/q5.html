A data type defines the type of value a variable can hold a value. two types of data types in javascript 1. Primitive data types 1. String 2. Number 3. Boolean 4. Null 5. Undefined 6. Symbol 7. BigInt 2. Non-primitive data types 1. Object 2. Array Primitive data types are immutable (cannot be changed) and are stored directly in the variable. Non-primitive data types are mutable (can be changed) and are stored as references.
